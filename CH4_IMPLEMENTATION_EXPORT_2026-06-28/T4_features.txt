NAVIGATION NOTE: the app is a single-screen STEP WIZARD (state machine in App.tsx),
NOT react-navigation. grep for Stack.Screen / createStackNavigator => NOT FOUND.
The "screens" are wizard steps switched by the `step` state:
  SPOT(0 distance) -> WINDOW(1 size) -> LIGHT(2 lux) -> FACING(3, optional) -> RESULTS(4)
Navigation gating is `maxReachable` / `canContinue` (App.tsx:549-562).

REAL USER-FACING CAPABILITIES (each maps to a built feature):
  - Measure plant-to-window distance via AR, or enter it from a tape (Step 1).
  - Measure window width/height/sill via AR or type them; skippable w/ logged reason (Step 2).
  - Capture spot light (lux) over a 10 s guided reading with plateau detection (Step 3).
  - Capture window facing via compass (optional) -> true north via GPS declination (Step 4).
  - Get a ranked, explainable recommendation list with per-plant score breakdown,
    direct-sun estimate, and a display-only light-science (PPFD/DLI) reference panel (Step 5).
  - Log an evaluation row (AR/tape, lux, UT383 refs) and export/share the CSV.
